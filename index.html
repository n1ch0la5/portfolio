About this site

  • Host: Digital Ocean
  • Server: Centos 6.4 64-bit Linux vps
  • LEMP stack: Nginx, PHP, Mysql

My goal with this site was to create a simple, one page, responsive site to serve as my digital portfolio and resume.

Take a look around the source code on github.

I'm not a world class designer so I didn't set out to reinvent the wheel. I wanted something clean and modern with a splash of color, so I took great inspiration from other well designed portfolio's and made it my own using the Foundation frontend framework.

Thanks for checking it out and be sure to let me know what you think!
